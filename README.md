🌐 Navegue master 
Um navegador web leve e rápido, focado em modularidade e eficiência. Realiza buscas integradas, exibe as principais notícias, vídeos e a previsão do clima.

Funcionalidades
Busca Web: Integração limpa com Wikipedia e Google.
Notícias: Feed das principais notícias (Google News) adaptado para o idioma local.
Vídeos: Atalho direto para buscas no YouTube.
Clima: Previsão do tempo local em tempo real.
Instalável (PWA): Pode ser adicionado à tela inicial do celular como um aplicativo nativo.
Tecnologias
HTML5, CSS3, JavaScript (Vanilla)
Service Workers e Web Manifest para PWA. Navegue Navegador web leve em um único arquivo HTML, criado para uso online via GitHub Pages.
Como usar Faça upload do arquivo index.html para seu repositório Ative GitHub Pages em Settings > Pages Acesse pelo link gerado Recursos Busca com resumo da Wikipedia Links diretos para Google, DuckDuckGo e Bing Aba Notícias com imagens (Google News via RSS) Aba Vídeos (YouTube) Aba Clima (wttr.in) Não requer cadastro, chave de API ou servidor.

APIs públicas utilizadas Este projeto consome apenas APIs gratuitas e públicas:

Wikipedia REST API (https://pt.wikipedia.org) rss2json (conversor de RSS do Google News) wttr.in (previsão do tempo) Links diretos para buscadores O uso dessas APIs é permitido para projetos pessoais e educacionais, conforme os termos de cada serviço.
